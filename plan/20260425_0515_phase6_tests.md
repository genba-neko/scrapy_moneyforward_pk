### 5. 実行 + 修正

#### 初回 collection エラー
日本語を `b"..."` バイトリテラルへ直接記述
→ Windows CP932 で SyntaxError "bytes can only contain ASCII"
→ `"...".encode("utf-8")` に変更で解消

#### pyright 11 errors (Phase 6 検証中)
- `crawler.stats` Optional でメンバアクセス不可 (× 8)
- `boto3.resource(...).Table(...)` の `Table` 不明 (× 1)
- `past_months: int | None` の `range(...)` 不正 (× 1)
- `Generator[Never, Any, None]` not iterable (× 1)

#### 修正
- 基底に `_inc_stat(key, count)` ヘルパ追加 → 全 spider で `self._inc_stat(...)` 経由
- pipeline 側は `getattr(spider.crawler, "stats", None)` ガード
- `boto3` の `Table` 取得行に `# type: ignore[attr-defined]`
- transaction.py で `months = self.past_months if self.past_months is not None else 12`
- `after_login` のデフォルトを `return iter(())` (型推論を明確化)
